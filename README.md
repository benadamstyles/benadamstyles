The following tasks can be triggered by running `yarn maid <command>`, where `command` is the title below.


```sh

```

### start

```sh
react-static start
```

### stage

```sh
react-static build --staging
```

### build

```sh
react-static build
```

```sh
serve dist -p 3001
```
